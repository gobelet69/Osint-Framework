Gobelet's OSINT Framework

A curated collection of tools and resources for Open Source Intelligence (OSINT). This framework is organized into categories to help streamline investigations.

Categories

1. Name Search

  • What's My Name - Identify usernames across multiple platforms.
  • Sherlock - Find social media accounts by username.
  • IntelTechniques Username Search - Search for usernames across various platforms.

2. Web Analysis

  • Web Check - Analyze websites for OSINT.

3. Image Search

  • TinEye - Reverse image search.
  • Google Images - Standard reverse image search.
  • Yandex Images - Powerful reverse image search with global reach.
  • PimEyes - Face search engine for finding similar faces.

4. Image Analysis

  • Photopea - Online photo editing and analysis.
  • Exif Tool - Extract metadata from images.
  • Photo Forensics - Analyze image tampering and inconsistencies.
  • Hex Editor - Inspect image files at the binary level.
  • Jimpl - Simplified image analysis.

5. Localization Search

  • Overpass Turbo - Query OpenStreetMap data for geospatial analysis.
  • Google Maps - Explore and investigate locations.
  • Google Earth Pro - Advanced geospatial analysis.
  • Osintgram - OSINT tool for Instagram data extraction.

6. Google Dorks

  • Google Hacking Database - Predefined Google search queries to find sensitive information.
  • Dork Generator - Generate Google dorks for specific queries.
  • DorkGenius - Another tool for generating Google dorks.

7. Cryptography

  • CyberChef - A Swiss army knife for data manipulation.
  • Cryptii - Encode, decode, and convert data.
  • OpenSSL - Toolkit for SSL/TLS and cryptography.
  • Cryptocrack - Software for breaking ciphers.

8. People Finder

  • Hunter.io - Find email addresses.
  • Maltego - Graphical link analysis tool.
  • Epieos - Search people by email and other identifiers.
  • Prospeo - Domain-based people search.
  • Osintgram - Instagram analysis and scraping.
  • emailGuesser - Tool to guess possible email addresses based on name patterns.

9. Advanced Search

  • X Advanced Search - Search engine for social media (formerly Twitter).
  • Web Archive - View archived versions of websites.
  • Google Advanced Search - Fine-tune search queries for better results.

10. File Analysis & Reverse Engineering

  • Binwalk - Analyze and extract data from binary files.
  • Ghidra - Open-source software reverse engineering suite.
  • IDA Free - Interactive Disassembler for reverse engineering.
  • Dogbolt - Online compiler explorer.
  • Hex Editor - Analyze and edit binary files.

Other Frameworks

  • Bellingcat OSINT Toolkit - Comprehensive OSINT guide and tools.
  • OSINT Framework - Categorized collection of OSINT tools.
